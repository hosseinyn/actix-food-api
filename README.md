# Actix food api

*Just a simple actix crud food api*

#### How to use?

1-Clone the repository:
`git clone https://github.com/hosseinyn/actix-food-api.git`

2- Open project directory
`cd actix-food-api`

3- Build the project using cargo
`cargo build`

4- Run the project using cargo :
`cargo run`
